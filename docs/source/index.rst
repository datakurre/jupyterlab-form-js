
jupyterlab_form_js
=====================================

Version: |release|

A Custom Jupyter Widget Library for bpmn.io/form-js


Quickstart
----------

To get started with jupyterlab_form_js, install with pip::

    pip install jupyterlab_form_js

or with conda::

    conda install jupyterlab_form_js
